### The Git config command

```
git config --global user.name "User name" 

git config --global user.email  "Email address"   

git init Test

git clone URL 

git add Filename  

git add*  

git commit -m " Commit Message"  

git status  

git push [variable name] master 

git push --all  

git pull URL  

git branch  

git merge BranchName  

git log 

```
